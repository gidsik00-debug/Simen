Expense Tracker (Трекер расходов)

Создать GUI-приложение «Expense Tracker» для отслеживания личных расходов с фильтрацией, подсчётом суммы за период, JSON и Git.

Пошаговая инструкция:

1. Создайте форму с полями:

○ Сумма

○ Категория (например, еда, транспорт, развлечения)

○ Дата

2. Добавьте кнопку «Добавить расход», которая добавляет запись в таблицу.

3. Реализуйте подсчёт суммы расходов за выбранный период.

4. Добавьте фильтрацию:

○ По категории

○ По дате

5. Сохраняйте и загружайте данные через JSON.

6. Проверяйте корректность ввода:

○ Сумма должна быть положительным числом.

○ Дата — в правильном формате.

7. Используйте Git:

○ Создайте репозиторий.

○ Добавьте .gitignore.

8. Напишите README (в формате .txt, .md или .docx), в котором укажите:

○ Имя и фамилию автора

○ Описание программы

○ Как использовать

○ Примеры использования (возможные тесты — по желанию)
